# Checklist de Deploy

## Antes do deploy
- [ ] issue analisada e compreendida
- [ ] correção revisada
- [ ] testes executados
- [ ] impacto mapeado
- [ ] backup ou plano de rollback definido

## Durante o deploy
- [ ] janela de publicação confirmada
- [ ] responsáveis avisados
- [ ] logs e métricas sendo acompanhados

## Depois do deploy
- [ ] funcionalidade validada
- [ ] erros monitorados
- [ ] equipe comunicada
- [ ] changelog atualizado
- [ ] evidências registradas
