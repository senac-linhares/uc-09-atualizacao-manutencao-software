# Exemplos de manutenção alinhados com a stack da turma

Os exemplos desta pasta foram organizados para praticar **correção de bugs**, **análise de causa raiz** e **melhorias de código** usando tecnologias já estudadas pelos alunos.

## Tecnologias contempladas
- `C# API`
- `MySQL`
- `Angular`
- `Flutter`
- `HTML`, `CSS`, `JavaScript` e `TypeScript`

## Como usar
1. Abra um exemplo.
2. Identifique o comportamento incorreto.
3. Registre a hipótese da causa.
4. Corrija o código.
5. Documente a mudança com `commit` claro.

> Os trechos contêm **bugs intencionais** para prática em sala.
