# Aula 09 — Testes e validação de correções

## 🎯 Objetivo
Verificar se a correção realmente resolveu o problema sem quebrar outras partes do sistema.

## Conteúdos
- testes manuais e automatizados;
- caso de teste;
- regressão;
- evidência da validação.

## Exemplo prático
Após corrigir um cálculo, a equipe cria testes para garantir que valores limites continuem funcionando.

## Atividade rápida
Escrever 3 casos de teste para um bug já corrigido.

## Fechamento
Teste é a prova de que a atualização ficou mais confiável.
