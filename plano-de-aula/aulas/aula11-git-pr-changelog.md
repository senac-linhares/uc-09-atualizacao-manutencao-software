# Aula 11 — Git avançado, PR review e changelog

## 🎯 Objetivo
Usar versionamento para registrar e revisar alterações com mais profissionalismo.

## Conteúdos
- `branch` por tarefa;
- mensagens de `commit` claras;
- `pull request`;
- revisão de código;
- changelog simples.

## Exemplo prático
Criar uma branch `fix/calculo-desconto`, registrar a correção e abrir um PR com descrição objetiva.

## Atividade rápida
Escrever 3 mensagens de `commit` boas e 3 ruins, explicando a diferença.

## Fechamento
Git não serve só para guardar código: ele também conta a história da manutenção.
