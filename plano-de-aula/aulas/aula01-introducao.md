# Aula 01 — Introdução à manutenção de software

## 🎯 Objetivo
Compreender o que é manutenção de software e por que ela é essencial depois que um sistema entra em produção.

## Conteúdos
- conceito de manutenção;
- tipos de manutenção;
- diferença entre desenvolver e manter;
- impacto no dia a dia das empresas.

## Explicação simples
Criar um sistema é só o começo. Depois que ele vai para uso real, aparecem bugs, pedidos de melhoria e mudanças de regra. É aí que entra a manutenção.

## Exemplo prático
Imagine um app de pedidos que calcula o frete errado. O sistema já funciona, mas precisa ser corrigido sem parar o negócio.

## Fechamento
**Ideia-chave:** software bom não é o que nunca muda, e sim o que consegue evoluir com segurança.
